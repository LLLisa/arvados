## Arvados Workbench 2

##### Setup
<pre>
brew install yarn
yarn install
</pre>
Install [redux-devtools-extension](https://chrome.google.com/webstore/detail/redux-devtools/lmhkpmbekcpmknklioeibfkpmmfibljd)

##### Start project
<code>yarn start</code>

##### Run tests
<code>yarn install</code>
<code>yarn test</code>

##### Production build
<code>yarn install</code>
<code>yarn build</code>
